# Food Point

Простой React сайт с надписью "food point" в центре экрана.

## Как запустить

1. Установить зависимости:
```bash
npm install
```

2. Запустить сервер разработки:
```bash
npm run dev
```

3. Открыть в браузере: http://localhost:5173/

## Как собрать для продакшена

```bash
npm run build
```

## Технологии

- React 18
- Vite 4
- CSS3 (градиенты, flexbox)

